# Маркдекс — материалы презентации

Папка для конференций и защиты: бренд **Маркдекс**, слоган **«Делаем веб лучше»**, **13 слайдов** (как в макете «Попов презентация»).

**Порядок и нумерация.** Номера слева в Figma — это порядок показа; **слайд с основным «спасибо» / итогом не обязан быть последним**: после него идут **дополнительные материалы** (техническая зрелость, сравнение, статус). В докладе это нужно **озвучить**: после основного заключения — коротко по дополнительным слайдам, затем вопросы. Подробно — в начале `presentation.md` и в тексте доклада к слайдам 10–13.

| Файл | Назначение |
|------|------------|
| `presentation.md` | Содержание слайдов 1–13, логика «основная линия → заключение → доп. материалы → Q&A», таблицы, **текст доклада от первого лица** |
| `index.html` | Веб-презентация: слайды 1–9, затем **10 — основное заключение**, **11–13** с пометкой «Дополнительные материалы»; тёмная тема, слайд демо — светлый блок; шрифт **Inter** |

Откройте `index.html` в браузере: навигация **стрелками**, **пробелом**, свайпом на тач-экране. Печать в PDF — через меню браузера.

Скриншоты из Figma подставьте в слайд демо вместо текстового-placeholder; год на титуле и формулировки про модель ИИ обновите под актуальный продукт.

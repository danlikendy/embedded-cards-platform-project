# Материалы выпускной квалификационной работы

## Структура

- **sokrashhenija-i-oboznachenija.md** — перечень сокращений и обозначений
- **vvedenie.md** — введение
- **glava1.md** — Глава 1 (теоретические основы)
- **spisok-literatury.md** — список использованных источников

## Рисунки (HTML)

Папка `figures/` содержит HTML-файлы рисунков в едином стиле:

| Файл | Описание |
|------|----------|
| fig1-tipologiya-platform.html | Рисунок 1 – Типология веб-платформ для персонального управления контентом |
| fig2-shema-mdx.html | Рисунок 2 – Схема обработки MDX |
| fig3-shema-agregacii.html | Рисунок 3 – Поток данных при агрегации метаданных |
| fig4-formaty-metadannyh.html | Рисунок 4 – Распределение форматов метаданных |

Общий стиль задаётся файлом `vkr-figures.css`. Для вставки в документ откройте HTML в браузере и выполните экспорт в изображение (скриншот) или скопируйте содержимое.

## Оформление

- В разделах 1.1–1.3 ссылки на библиографию не используются (по требованию)
- Таблицы и рисунки упоминаются в тексте до их появления
- Стиль рисунков: приглушённая палитра, без ярких красок

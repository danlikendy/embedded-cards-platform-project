# Материалы выпускной квалификационной работы

## Структура

- **sokrashhenija-i-oboznachenija.md** — перечень сокращений и обозначений
- **vvedenie.md** — введение
- **glava1.md** — Глава 1 (теоретические основы; глоссарий и трассировка в конце п. 1.3)
- **glava2.md** — Глава 2 (п. 2.1–2.3; табл. 4–12: сравн. анализ, PESTLE, RACI, SIPOC, трассировка, риски, TCO, продуктовые метрики; рис. 7–10)
- **glava3.md** — Глава 3 (п. 3.1–3.4; табл. 13–17, рис. 11 — C4-контейнеры, рис. 12 — схема редактора, рис. 13 — поток агрегации, рис. 14 — UMUX-Lite)
- **spisok-literatury.md** — список использованных источников

## Оформление

- В разделах 1.1–1.3 ссылки на библиографию не используются (по требованию)
- Таблицы и рисунки упоминаются в тексте до их появления
- Стиль рисунков: приглушённая палитра, без ярких красок

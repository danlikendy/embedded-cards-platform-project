## Приложение: вероятные вопросы комиссии

**Q: Почему не использован готовый сторонний API (Microlink, Iframely)?**  
A: Стоимость для 10 000 пользователей — $59-120/мес (5 439-11 058 руб.) против 922 руб./мес собственного Upstash Redis. Дополнительно: собственный агрегатор даёт поле excerpt, которого сторонние сервисы не предоставляют.

**Q: Почему выборка UMUX-Lite только 7 человек?**  
A: Finstad (2010) обосновывает валидность UMUX-Lite на выборке 5-8 для экспертного тестирования. Расширение до ≥ 30 (полная SUS) запланировано как перспектива развития.

**Q: Обоснование ставки дисконтирования 28%?**  
A: Ключевая ставка ЦБ РФ на 26.04.2026 = 16% + 12 п.п. премии за риск стартапа (верхний предел диапазона методических указаний). Даже при r=40% NPV остаётся положительным (+1 028 тыс. руб.).

**Q: Как обеспечивается 152-ФЗ при коммерциализации?**  
A: Миграция на Yandex Cloud (4 800 + 250 + 1 450 + 500 = 7 000 руб./мес). Премия ∆TCO₁₅₂ = 50 820 руб./год = 5,08 руб./польз./год. Это не меняет тариф Pro.

**Q: Почему хранение в localStorage, а не в БД?**  
A: MVP-решение, достаточное для академической валидации. Переход на облачное хранилище зафиксирован в технологической перспективе (раздел 3.6.4).

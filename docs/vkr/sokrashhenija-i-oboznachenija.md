# ПЕРЕЧЕНЬ СОКРАЩЕНИЙ И ОБОЗНАЧЕНИЙ

**API** — Application Programming Interface (интерфейс прикладного программирования)

**CMS** — Content Management System (система управления контентом)

**CRDT** — Conflict-free Replicated Data Type (конфликтно-свободный реплицируемый тип данных)

**DOM** — Document Object Model (объектная модель документа)

**ESM** — ECMAScript Modules (модули ECMAScript)

**JSON-LD** — JavaScript Object Notation for Linked Data (JSON для связанных данных)

**JSX** — JavaScript XML (расширение синтаксиса JavaScript для описания структуры интерфейса)

**MDX** — Markdown + JSX (гибридный формат разметки, объединяющий Markdown и JSX)

**MDAST** — Markdown Abstract Syntax Tree (абстрактное синтаксическое дерево Markdown)

**OG** — Open Graph (открытый граф; протокол метаданных для социальных сетей)

**OT** — Operational Transformation (операционное преобразование)

**PWA** — Progressive Web Application (прогрессивное веб-приложение)

**REST** — Representational State Transfer (передача репрезентативного состояния)

**RDFa** — Resource Description Framework in Attributes (фреймворк описания ресурсов в атрибутах)

**SEO** — Search Engine Optimization (оптимизация для поисковых систем)

**SPA** — Single Page Application (одностраничное приложение)

**SUS** — System Usability Scale (шкала оценки удобства использования системы)

**UMUX-Lite** — Usability Metric for User Experience Lite (облегчённая метрика удобства пользовательского опыта)

**WYSIWYG** — What You See Is What You Get (визуальное редактирование «как видишь, так и получаешь»)
